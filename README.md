# Задание
Представим, что мы создаем игру по мотивам **StarWars**. 

"Заказчик" в качестве первых результатов просит создать 
**главную страницу** и **окно создания** персонажа (текущее хранение данных в local storage / indexed DB).

При создании персонажа использовать данные из открытого API https://swapi.tech/.

В рамках задачи допускается "занаучивание" и усложнение кода, с целью показать владение теми или иными паттернами фронтовой разработки.

Мы не проекте используем следующий стек: **vite, tailwindcss, axios, useQuery**. Использование этих библиотек не обязательно, 
но приветствуется.
